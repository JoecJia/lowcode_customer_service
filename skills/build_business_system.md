# Skill: 搭建具体业务系统指南 (Business System Building Guide)

## 描述
用于引导用户在低代码平台上一步步搭建完整的、具有特定行业特征的业务系统（如：学工管理系统、实习系统等）。

## 输入参数
- `system_type` (string): 目标系统类型（如“学工系统”）。
- `detail_level` (string): 指南的详细程度。

## 执行逻辑
1. **调用指南库**：在 `skills/build_business_system.md` 或相关 context 中检索该类系统的标准建设方案。
2. **步骤拆解**：
   - 第一步：数据建模（建哪些表）。
   - 第二步：流程配置（设置哪些节点）。
   - 第三步：权限划分（谁能看，谁能改）。
3. **最佳实践提示**：提供避坑指南和效率提升建议。

## 使用场景
当用户明确提出“帮我搭建一个XX系统”或“XX系统的建设方案是什么？”时调用。
