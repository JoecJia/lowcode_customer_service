# Skill: 产品功能使用方法 (Product Feature Usage)

## 描述
专门用于回答用户关于“超星低代码”产品中具体功能如何操作、如何配置、按钮位置、参数设置等“如何做”的问题。

## 输入参数
- `feature_name` (string): 用户咨询的具体功能名称（如：零代码建表、流程设计器、权限设置等）。
- `query` (string): 用户的具体疑问点。

## 执行逻辑
1. **调用知识检索**：在 `context/product_docs/` 和 `context/faq/` 中检索该功能的操作指南。
2. **结构化回复**：按照“操作路径 -> 关键设置 -> 注意事项”的结构回复用户。
3. **视觉引导**：如果检索内容中包含图片标签，按需返回图片链接以辅助说明。

## 使用示例
- 用户问：“怎么设置表单的必填项？”
- Skill 动作：检索“必填项设置”，返回操作步骤。
