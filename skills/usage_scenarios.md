# Skill: 产品功能使用案例 (Product Feature Usage Cases)

## 描述
用于回答用户关于某个功能“在什么情况下使用”、“如何解决业务问题”的问题，侧重于提供具体的应用案例和实现逻辑。

## 输入参数
- `feature_name` (string): 涉及的功能名称。
- `user_context` (string): 用户的业务背景（如果有）。

## 执行逻辑
0. **需求预审**：评估用户提供的信息是否明确。若缺失功能名称或背景模糊，调用 `clarifying_questions` (信息补充与反问) 技能获取必要信息。
1. **匹配场景库**：通过调用 `knowledge_retrieval` 技能，在 `context/product_docs/` 中检索该功能对应的典型应用案例和场景描述。
2. **功能实现逻辑阐述**：详细说明此功能是如何通过具体的配置或操作，来对应并解决上述业务场景中的问题的。
3. **关联推荐**：根据当前场景，推荐配合使用的其他功能，并描述这些功能如何满足用户的进一步需求。

## 使用示例
- 用户问：“数据工厂适合在什么场景下使用？”
- Skill 动作：检索并返回“考试试卷分数处理”、“排版课表”等案例，解释如何通过数据工厂节点进行加工，并推荐配合仪表盘进行展示。
