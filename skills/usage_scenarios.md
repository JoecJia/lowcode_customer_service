# Skill: 产品功能使用场景 (Product Feature Usage Scenarios)

## 描述
用于回答用户关于某个功能“在什么情况下使用”、“有什么好处”、“能解决什么业务问题”的问题，侧重于功能价值和适用边界。

## 输入参数
- `feature_name` (string): 涉及的功能名称。
- `user_context` (string): 用户的业务背景（如果有）。

## 执行逻辑
1. **匹配场景库**：在 `context/product_docs/` 中检索该功能对应的典型应用案例和场景描述。
2. **价值阐述**：说明该功能如何简化工作流或提高效率。
3. **关联推荐**：根据场景推荐配合使用的其他功能。

## 使用示例
- 用户问：“AI填充功能适合在什么场景下使用？”
- Skill 动作：返回“大批量数据录入”、“历史表单快速迁移”等应用场景。
