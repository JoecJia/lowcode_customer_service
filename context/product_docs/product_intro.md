# 产品名称：超星低代码平台

## 平台包含以下产品：表单引擎、审批引擎、仪表盘引擎（当前项目中还叫图表引擎）、信息查询引擎、数据工厂、聚合表、业务流引擎、报表引擎、报告引擎、表单关联图谱、分类树

## 各产品介绍：
- **表单引擎**：一种万能表单类应用生成工具，兼具数据收集、管理与输出功能。提供丰富的字段类型与属性设置，支持自定义样式、提交校验及数据导入导出。
- **审批引擎**：用于处理复杂的业务审批流程，支持多级、并行及分支审批设计。通过与表单深度集成，实现业务审批的规范化与自动化管理。
- **仪表盘引擎（或图表引擎）**：对表单、审批及数据工厂的数据进行深度处理与分析，通过指标图、柱状图、饼图等多种可视化组件，直观展示数据分布、趋势与异常。
- **信息查询引擎**：信息查询可以帮助用户从表单、审批和上传的 excel 中快速准确地查询到所需要的信息。
- **数据工厂**：支持将表单和审批作为数据源，可以将表中的数据进行统计分析，生成想要的统计结果，并将结果返回到某个表单应用里。
- **聚合表**：针对有限额的报名、申请、售卖等场景，聚合表可以防止超出限额的情况发生。
- **业务流引擎**：一款实现数据自动化流转的工具。通过查询、操作数据、分支及连通器等节点的自由组合，实现多表之间及表与外部服务间的数据互联互通。
