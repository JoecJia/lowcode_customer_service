# 公式编辑使用指南 (Formula Editing Guide)

## 一、公式编辑概述
公式编辑是存在于单行输入、多行输入、数字输入、日期中的一种特殊的默认值赋予方式。它能够获取当前填写页的数据，并通过配置的函数进行一系列计算后，将结果返回到对应字段中。

## 二、使用流程
1. **添加字段**：在表单中添加单行输入、多行输入、数字输入或日期字段。
2. **选择默认值**：在右侧字段属性的“默认值”中选择“公式编辑”。
3. **配置公式**：在弹出的配置窗口中组合可用变量（表单字段）和函数。
4. **实时计算**：在填写页实现自动计算和赋值。

## 三、核心概念
### 1. 函数的输入与输出
- **有输入输出**：大部分函数需要输入参数（如 `AVERAGE` 需要输入成绩）。
- **无输入输出**：部分函数无需输入（如 `NOW()` 获取当前时间，`GETFID()` 获取单位 FID）。

### 2. 参数类型
- **文本**：单行输入、多行输入、单选等。
- **数字**：数字输入、滑动条、计算公式等。
- **时间戳**：日期、日期区间。
- **数组**：多选、下拉复选、子表单字段。
- **特殊参数**：定位字段（仅用于 `DISTANCE` 函数）。

### 3. 触发条件
- **变量触发**：当公式包含本表单变量时，修改变量值会触发重新计算。
- **进入触发**：公式不包含变量时，进入表单即触发。
- **注意**：表单预览不触发；编辑数据时若不修改变量，保持原值不触发。

## 四、注意事项
- **英文字符**：仅可使用英文字符和数学运算符（`==`, `!=`, `<`, `+`, `-`, `*`, `/` 等）。
- **生效范围**：仅在表单填写、后台添加/编辑数据时生效。
- **数字比较**：使用 `==` 判断数字相等。
- **文本比较**：使用 `EXACT` 或 `EXACTALL` 函数。
- **类型匹配**：公式返回值必须符合目标字段类型。
- **与计算公式的区别**：公式编辑支持复杂函数，而“计算公式”字段仅支持基础加减乘除且仅限数字字段。

## 五、函数参考手册

### 数学函数
- `ABS(数字)`: 获取绝对值。
- `AVERAGE(数字1, ...)`: 获取算术平均值。
- `SUM(数字1, ...)`: 获取总和。
- `MAX/MIN`: 获取最大/最小值。
- `ROUND(数字, 位数)`: 四舍五入。
- `INT(数字)`: 取整。
- `MOD(被除数, 除数)`: 取余。
- `RANDOM()`: 返回 0-1 之间的随机数。
- `POWER(数字, 指数)`: 乘幂。
- `SQRT(数字)`: 平方根。

### 文本函数
- `CONCATENATE(文本1, ...)`: 合并文本。
- `EXACT(文本1, 文本2)`: 比较文本是否相同。
- `LEN(文本)`: 获取字符个数。
- `LEFT/RIGHT/MID`: 截取文本。
- `REPLACE`: 替换文本。
- `UPPER/LOWER`: 大小写转换。
- `SPLIT(文本, 分隔符)`: 分解为数组。
- `JOIN(连接符, 数组)`: 数组合并为文本。
- `TEXT(值, 格式)`: 转化为特定格式文本。

### 日期函数
- `DATE(时间戳)`: 转换为日期对象。
- `DATEDIF(开始, 结束, 单位)`: 计算日期差。
- `NOW()`: 获取当前时间。
- `YEAR/MONTH/DAY/HOUR/MINUTE`: 提取时间分量。
- `NETWORKDAYS`: 计算工作日。
- `EDATE(日期, 月数)`: 加减月份。

### 逻辑函数
- `IF(逻辑, 真值, 假值)`: 条件判断。
- `IFS(条件1, 值1, 条件2, 值2, ...)`: 多条件判断。
- `AND/OR/NOT/XOR`: 逻辑运算。

### 高级与集合函数
- `COUNT(值1, ...)`: 统计参数数量。
- `COUNTIF(数组, 条件)`: 统计满足条件的个数。
- `ARRAYGET(数组, n)`: 获取数组第 n 个值。
- `GETFID/GETFORMNAME`: 获取系统/表单信息。
- `UUID()`: 生成 32 位随机码。
- `MD5(文本)`: 生成 MD5 值。
