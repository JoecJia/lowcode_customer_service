# 图表引擎使用说明

### 图表引擎简介

#### 功能介绍

###### 使用图表引擎可以对表单引擎/审批引擎收集来的数据、数据工厂加工后的数据进行处理和分析，并将分析结果以多种图表形式展示出来。相比单纯的数字，图表可以让用户更容易洞察到数据的分布、趋势、关系以及异常点等。满足用户快速掌握数据反映的情况，快速找到数据背后隐藏的现实问题的需求。

###### 图表引擎中包含多种样式的组件，可以根据实际需要选择图表组件进行统计看板的搭建：

###### • 可以通过指标图、明细表格查看选定数据的汇总和明细；

###### • 可以通过数据管理表展示数据，并对有权限的数据进行添加、导入、导出、编辑、删除等操作；

###### • 通过柱状图、堆叠柱状图、条状图、堆叠条状图等，显示数据的分类对比结果；

###### • 通过折线图对数据进行统计，显示出数据的发展趋势；

###### • 通过饼状图体现数据中每个部分的比例；

###### • 通过雷达图直观地展现多维数据集，查看哪些变量具有相似的值、变量之间是否有异常值；

###### • 通过词云图组件展示词语出现的频率和重要性，帮助用户快速掌握重点内容。

###### 使用图表引擎创建的一个图表应用中可以添加多个图表，并且可以灵活调整图表的样式、大小、位置和布局等。

###### 创建完成的图表应用支持在PC端和移动端查看。
