# 数据工厂使用说明

低代码开发平台—-数据工厂培训

什么是数据工厂？为什么会有数据工厂？

就是对数据进行加工处理的工具

通过表单构建的业务系统，完成了数据的收集、管理、权限的分配等一系列过程后，如何处理、利用、分析这些数据，得到符合客户个性化需求的结果。

目前有八个数据节点，横向连接、追加合并、数据筛选、分组汇总、字段设置、字段排序、去重、行转列

应用场景：考试试卷分数处理，教研积分处理、排版课表、评价统计

数据流

简介

功能简介

在数据工厂里，您也可以创建多个数据流，实现不同的数据加工和处理流程。

数据工厂是可以加工处理数据的场所，而其中的数据流则承担着具体的数据加工和处理工作。

应用场景

数据流可以选择多个表单作为数据源，然后自由组合数据处理节点，实现复杂的多表数据的处理，得到输出表。目前有 8大数据处理节点：

横向连接：将两张表左右相连，连接为一张新增了多列数据的新表。

例如：第一张表是学生数学成绩，第二张表是语文成绩，需要将学生所有成绩统计到一张表，用横向链接

追加合并：将多张表上下相连，连接为一张新增了多行数据的新表。

例如：第一张表1班学生成绩，第二张表2班学生成绩，行字段不一致，列一致，可以追加合并连接到一张表单中

数据筛选：添加筛选条件，过滤掉无用数据，只保留需要处理的数据进行加工计算。

分组汇总：分组汇总是指对原始数据根据条件，将相同的数据先合并到一组，然后按照分组后的数据进行汇总计算。

字段设置：字段设置可以设置表单字段显示/隐藏、排序（字段显示顺序）、重命名，也可以添加新的计算字段、排名字段。

字段排序：字段排序按单张表中的指定字段进行排序操作（内容排序）。

去重：对重复数据保留最新一条

行转列：一维表转化为二维表

使用规则：

每个数据流中，支持添加多个输入节点。即可以对多个数据源表单进行处理和计算。

输入节点不能直接跟输出节点相连，中间需要插入至少 1 种数据处理节点。

如果表单中有多个子表单，只能选择 1 个子表单的数据进行输入。如果需要用到多个子表单的数据，可以添加多个输入节点。

连接方式

连接方式目前支持内连接、左连接和右连接。横向连接时，是根据连接字段进行匹配的，连接字段相同的数据根据连接方式进行左右拼接。比如，内连接，取两张表的数据交集，连接字段只有在左右两张表都包含才会进行匹配，否则舍弃。

分组汇总其实就是先分组再汇总。
